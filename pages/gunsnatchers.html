Gunsnatchers

Gunsnatchers is a fast paced top down shooter about thinking on your feet, and clearing levels of enemies as fast as possible.

For this project, I used a roguelike code structure I had been working on in the past. It includes all the basic top-down combat elements you'd expect in a roguelike game but highly customizable. There are weapons with tweakable stats and room for custom code, customizable enemy behavior using inheritance, status effects, and a working player inventory and weapon system.

I also experimented with basic AI using state machines to make the enemies feel as alive as possible.
